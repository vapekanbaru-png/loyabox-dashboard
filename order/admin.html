Google Sheets Setup
Cara setup:
1. Buka sheets.google.com → buat sheet baru
2. Extensions → Apps Script → paste isi file vpknbr-apps-script.gs
3. Deploy → New Deployment → Web App
4. Execute as: Me | Access: Anyone
5. Copy URL → paste di atas → Simpan
